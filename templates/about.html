About This Project

This Threat Detector is a machine learning-based web app that identifies cyberbullying and harmful messages. By analyzing the text input, it classifies messages as safe or potentially harmful.

The tool was developed with the goal of raising awareness and offering immediate feedback in potentially harmful online interactions.

Built with: Flask, Scikit-learn, HTML, CSS

Creator: Abhirama
